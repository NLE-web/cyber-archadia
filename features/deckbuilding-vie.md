# Deckbuilding "Vie"
Statut : <span style="color:red">Pas implémenté</span>

## Description
Mécanique de construction de deck représentant les expériences, les traumatismes et les capacités du personnage.

## Fonctionnement
*   **Deck de cartes** : Chaque personnage possède un deck unique.
*   **Actions** : Piocher des cartes, afficher le contenu du deck, consulter la défausse.
*   **Limites** : Le joueur peut piocher jusqu'à un maximum de X cartes (configurable) plus les bonus accordés par d'autres cartes.
*   **Gestion MJ** : Les cartes sont créées via l'interface MJ et assignées au deck du joueur par le MJ.
