# Génération de DEEPDIVE
Statut : <span style="color:red">Pas implémenté</span>

## Description
Outil spécialisé pour le MJ permettant de créer des structures de réseaux pour la feature Netrunning.

## Fonctionnalités
*   **Générateur de Graphes** : Création automatique ou manuelle de réseaux avec différents types de nœuds.
*   **Placement d'ICE** : Ajout de protections (Intrusion Countermeasures Electronics) sur les nœuds sensibles.
*   **Récompenses & Données** : Assignation de fichiers ou de contrôles système aux nœuds terminaux.
