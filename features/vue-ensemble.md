# Vue d'ensemble (Console MJ)
Statut : <span style="color:red">Pas implémenté</span>

## Description
Tableau de bord central pour le Maître du Jeu permettant de suivre l'état de tous les personnages joueurs en temps réel.

## Informations affichées
*   **Santé & Stress** : Barres de vie et niveaux de stress de chaque joueur.
*   **Actions rapides** : Boutons pour infliger des dégâts, soigner ou modifier le stress.
*   **Accès Fiches** : Liens directs pour ouvrir la fiche détaillée de chaque personnage.
*   **Statut de session** : Liste des joueurs connectés et actifs.
