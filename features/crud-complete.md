## Description
Gestion administrative complète de la base de données via une interface dédiée dans la console MJ.

## Éléments gérés
*   **Objets & Équipement** : Création et modification des items (armes, armures, etc.).
*   **Capacités & Talents** : Base de données des aptitudes disponibles.
*   **Contacts** : Gestion des PNJ et des relations sociales.
*   **Sorts/Programmes** : Pour le Netrunning et autres capacités spéciales.
*   **Données Monde** : Lieux, factions, et autres éléments de lore.
