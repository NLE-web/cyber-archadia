# Interface de création de cartes
Statut : <span style="color:red">Pas implémenté</span>

## Description
Outil réservé au Maître du Jeu (MJ) pour concevoir de nouvelles cartes destinées au système de deckbuilding "Vie".

## Fonctionnalités
*   **Éditeur de carte** : Définition du nom, de l'illustration, du texte de règle et des effets mécaniques.
*   **Catégorisation** : Classement des cartes par type ou rareté.
*   **Assignation** : Interface pour envoyer directement une carte créée dans le deck d'un ou plusieurs joueurs.
