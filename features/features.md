# Fonctionnalités - Cyber Archadia

Ce document répertorie les fonctionnalités prévues pour l'application Cyber Archadia.

## Fiche de personnage
* **Inventaire** : Gestion des objets, armes et équipements possédés par le personnage.
* **Capacités** : Vue compilées des capacités activables et des tokens actuellement subis (bénéfique/malus).
* **Level up** : Système de progression et d'évolution des statistiques/talents.
* **Téléphone & gestion sociale** : Interface de communication en jeu et gestion des contacts.
* **Deckbuilding "Vie"** : Mécanique de construction de deck représentant les expériences et capacités du personnage.
* **Netrunning** : interface cyberdeck, DEEPDIVE dans des réseaux

## Console MJ (Maître du Jeu)
* **Vue d'ensemble** : Accès aux fiches de tous les personnages joueurs.
* **Interface de création de cartes** : Outil pour concevoir de nouvelles cartes pour le deckbuilding.
* **Interface CRUD complète** : Gestion administrative de la base de données (objets, talents, armes, contacts, sorts, etc.).
* **Génération aléatoire** : Génération de noms, lieux, pubs, intrigue, etc via Symfony AI
* **Génération de DEEPDIVE** : Générer des réseaux pour la feature DEEPDIVE
