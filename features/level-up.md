# Level up
Statut : <span style="color:red">Pas implémenté</span>

## Description
Système de progression permettant aux joueurs d'améliorer leurs personnages suite à l'acquisition d'expérience ou d'événements en jeu.

## Fonctionnalités
*   **Compétences** : Affichage de toutes les compétences existantes avec possibilité d'en créer de nouvelles spécifiquement pour le personnage.
*   **Progression** : Bouton "+" pour augmenter le niveau d'une compétence (représenté par des carrés jaunes).
*   **Achat de Talents** : Liste des dons, skills et talents achetables avec de l'XP.
*   **Intégration** : Les achats de talents ajoutent automatiquement de nouveaux éléments dans l'onglet "Capacités" du personnage.
