# Génération aléatoire (Symfony AI)
Statut : <span style="color:red">Pas implémenté</span>

## Description
Outil de support pour le Maître du Jeu utilisant l'intelligence artificielle pour générer du contenu à la volée.

## Éléments générables
*   **Identités** : Noms, prénoms, backgrounds pour les PNJ.
*   **Lieux & Ambiances** : Descriptions de bars, ruelles, appartements cyberpunk.
*   **Intrigues** : Accroches de scénarios ou complications imprévues.
*   **Contenus Publicitaires** : Slogans et publicités immersives pour l'univers.
