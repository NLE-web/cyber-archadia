# Téléphone & Gestion Sociale
Statut : <span style="color:red">Pas implémenté</span>

## Description
Interface immersive simulant un smartphone pour le personnage, permettant la communication avec les PNJ et la gestion du réseau social.

## Fonctionnalités
*   **Liste de Contacts** : Affichage des contacts avec image de profil, nom et courte description.
*   **Messagerie** : Système de chatbox pour envoyer et recevoir des messages.
*   **Notifications** : Indicateur visuel du nombre de messages non lus par contact.
*   **Interaction MJ** : Le MJ peut envoyer des messages en temps réel via les contacts créés dans la console.
